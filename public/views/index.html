Feathercoin

Production Version 0.9.6

Check out the Feathercoin GitHub release page for official download links.

Public Feathercoin API: Machine readable stats & blockchain info can be accessed directly through the REST and Websockets APIs.

Feathercoin bypasses the old banking system by using peer-to-peer technology. Payments are borderless and can be processed by anyone with affordable computer hardware using free software. This work is rewarded with new feathercoins which are issued by the network.

If you are new to Feathercoin, check out Feathercoin.com and the Feathercoin forum.
